# Issue Tracker

Use this file to tell agents how repository story packets relate to external
work trackers such as GitHub Issues, Linear, Jira, or another system.

## Current Tracker

- External tracker: none configured.
- Repository story packets are the canonical work record until this file says
  otherwise.

## Linking Rule

When an external issue exists, link it from the story packet. Do not rely on the
external issue as the only source of accepted scope, validation expectations,
decisions, or evidence unless the project explicitly grants agents reliable
access to that system.
