# Handoffs

This folder stores durable resume notes for unfinished or transferred work.

Use `docs/templates/handoff.md` when creating a handoff.

## Active Handoff Check

Before starting new work, agents should check this folder for active handoffs
unless the human explicitly says to ignore previous work.

An active handoff should identify:

- related story or packet
- current state
- files touched
- commands run
- single next action
- open risks
- evaluation status

## Naming

Suggested filename:

```text
HANDOFF-YYYYMMDD-story-or-topic.md
```
